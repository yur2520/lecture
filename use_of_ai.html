생성형 AI 효과적으로 활용하기

프롬프트 작성 요령

프롬프트(Prompt)는 생성형 AI가 작업을 수행하도록 유도하기 위해 사용자가 입력하는 질문, 명령, 또는 지시 사항을 의미합니다.

# 역할 부여하기 (Persona)

AI에게 특정 전문가 역할을 부여하면, 해당 관점에 맞는 깊이 있는 답변을 얻을 수 있습니다.

일반 과제 예시
당신은 교육과정 전문가야. 초등학교 5학년 사회 교과서의 '인권' 단원을 바탕으로 한 차시 분량의 토론 수업 계획안을 작성해 줘.
웹 개발 예시
너는 숙련된 프론트엔드 개발자야. '인권'을 주제로 한 초등학생용 학습 웹사이트의 메인 페이지를 만들어 줘.

# 명확하게 지시하기

모호한 표현 대신 '만들어 줘', '요약해 줘', '분석해 줘' 등 AI가 수행해야 할 행동을 명확히 지시하세요.

일반 과제 예시
초등학교 3학년 수준의 분수 개념 학습을 위한 객관식 문제 5개를 만들어 줘.
웹 개발 예시
분수 덧셈을 연습할 수 있는 간단한 웹 게임을 제작해 줘. 정답/오답 메시지가 뜨는 기능이 있어야 해.

# 맥락 및 사용자 수준 제공

누구를 위한 것인지(배경)와 질문하는 나의 수준(초보/전문가)을 알려주면 최적화된 맞춤형 답변을 받을 수 있습니다.

① 배경 상황 설명하기

일반 과제 예시
다음 주에 4학년 학생들과 '환경 보호' 캠페인을 할 거야. 학생들이 사용할 피켓 문구 아이디어를 줘.
웹 개발 예시
학생들이 환경 보호 실천을 기록하는 온라인 게시판을 만들고 싶어. 등록 버튼을 누르면 내용이 저장되는 기능이 필요해.

② 나의 수준(Level) 명시하기

일반 과제 예시
나는 코딩 초보 교사야. 엔트리로 '장애물 피하기' 게임 만드는 법을 아이들에게 가르칠 수 있게 쉬운 활동지를 만들어 줘.
웹 개발 예시
나는 자바스크립트 입문자야. 카운터 앱 코드를 짜주는데, 각 줄마다 어떤 역할을 하는지 주석으로 상세히 설명해 줘.

# 형식 지정 및 어조 설정

원하는 결과물의 형태(표, 리스트)와 글의 분위기(격식, 친근함)를 함께 지정하면 완성도가 높아집니다.

A. 형식(Format) 지정하기

일반 과제 예시
이번 주에 배울 과학 단원 핵심 용어 5개를 '용어', '쉬운 설명', '예시' 항목이 있는 표 형식으로 정리해 줘.
웹 개발 예시
과학 용어 퀴즈 웹페이지를 만들어 줘. 아코디언 UI 형식을 사용해서 용어 카드를 클릭하면 설명이 펼쳐지도록 해줘.

B. 어조(Tone) 설정하기

일반 과제 예시
새 학기를 시작하는 학생들에게 용기를 주는 환영사를 따뜻하고 희망적인 어조로 작성해 줘.
웹 개발 예시
우리 반 학급 홈페이지 대문을 꾸밀 거야. '꿈과 희망이 가득한 교실' 문구가 아기자기하고 따뜻한 디자인으로 보이게 CSS를 짜줘.

# 다양한 예시 제공

원하는 결과의 텍스트 샘플을 보여주거나, 가능한 경우 이미지 등의 파일을 직접 첨부하여 질문하면 AI가 훨씬 정확하게 이해합니다.

① 텍스트 예시 (Few-shot)

일반 과제 예시
다음 예시처럼, 시의 함축적 의미를 묻는 발문 3가지를 만들어 줘.
예시: '시인은 왜 하필 이 계절을 배경으로 설정했을까?'
웹 개발 예시
내가 보여주는 웹사이트(URL 또는 설명)와 비슷한 레이아웃 구조로 포트폴리오 페이지를 만들어 줘.

② 이미지 첨부 (멀티모달 기능)

일반 과제 예시
(교과서 도표 사진을 첨부하며)
이 도표를 해석해서 학생들이 이해하기 쉬운 줄글로 요약해 줘.
웹 개발 예시
(내가 그린 레이아웃 스케치를 첨부하며)
이 스케치대로 HTML/CSS 코드를 작성해 줘. 로고는 왼쪽, 메뉴는 오른쪽에 배치해 줘.

# 단계별로 나누기 (Chain of Thought)

복잡한 작업을 한 번에 요청하기보다 순서대로 처리하게 하면 성공 확률이 비약적으로 상승합니다.

일반 과제 예시
1. 먼저 '조선시대 신분제도' 학습 목표를 3개 정해줘.
2. 그 다음, 각 목표에 맞는 활동 아이디어를 하나씩 제안해 줘.
웹 개발 예시
1. 먼저 투두리스트 앱의 기본 HTML 구조를 짜줘.
2. 그 코드를 바탕으로 할 일 추가/삭제 자바스크립트 기능을 구현해 줘.

# 제약 조건 설정하기

안 되는 것과 꼭 포함해야 할 것을 명시하여 결과물의 범위를 좁혀줍니다.

일반 과제 예시
세종대왕 업적 설명문을 500자 내외로 쓰되, 반드시 '한글 창제'와 '과학 기술' 내용이 포함되어야 해.
웹 개발 예시
카드 맞추기 게임을 만들어 줘. 단, 카드는 총 12개(3x4)여야 하고, 뒷면 색상은 #a8dadc 로 지정해 줘.

# 영어로 질문하기 (English Prompts)

대부분의 생성형 AI는 방대한 영어 데이터로 학습되었습니다. 복잡한 코딩 문제나 논리적 추론이 필요할 때는 영어를 사용하거나 핵심 단어를 영어로 표기하는 것이 더 정확할 수 있습니다.

단어 섞어 쓰기 (Code Mixing)
React의 useState Hook을 사용해서 Counter Component를 만드는 예제 코드를 작성해 줘.

✨ 기술 용어(Term)를 영어로 쓰면 AI가 더 명확하게 인식합니다.

전체 영어 질문 (Web Dev)
Create a responsive navigation bar using HTML and Tailwind CSS.
It should have a logo on the left and links on the right.

# 환각 현상 (Hallucination)

AI가 거짓 정보를 그럴듯하게 만들어내는 현상입니다. 반드시 사실 여부를 교차 검증하세요.

검증 방법 예시
초등학교 5학년 과학 수업 지도안을 짜줘. 2022 개정 교육과정을 반영하고, 관련 교육부 고시 자료의 출처를 명시해줘.
최근 발표된 정책 내용을 요약하고, 공식 발표 자료의 원문 링크를 함께 제공해줘.

# 정보의 편향성

학습 데이터의 편견이 결과물에 반영될 수 있습니다. 비판적인 시각으로 검토하세요.

공정성 확보 프롬프트
학급 규칙을 정하려고 해. 특정 성별에 대한 고정관념이 드러나지 않도록 중립적인 표현을 사용해줘.
다문화 가정 학생들을 위한 자료야. 다양한 문화적 배경을 가진 학생들이 소외감을 느끼지 않도록 배려해서 작성해줘.

# 저작권 및 지적 재산권

AI 생성물의 저작권 문제는 여전히 논의 중입니다. 활용 시 출처를 밝히고 상업적 이용에 주의하세요.

저작권 보호 프롬프트
수업 자료에 쓸 삽화를 그려줘. 교육용 목적으로 자유롭게 사용할 수 있는 스타일로 생성해줘.
학습지를 만들 때 저작권에 문제가 없는 무료 이미지를 사용하거나 직접 그릴 수 있는 아이디어를 줘.

# 보안 및 개인 정보 보호

학생의 실명, 성적, 생활기록부 등 민감한 개인정보는 절대 AI에 입력하지 마세요.

잘못된 예시
우리 반 학생 명단과 성적 파일이야. 이걸로 가정통신문 써줘.
김철수 학생의 상담 내용을 붙여넣을게. 분석해줘.
올바른 사용
성적 안내 가정통신문 템플릿(양식)만 만들어줘. 내용은 내가 채울게.
일반적으로 교우 관계에 어려움을 겪는 학생을 상담할 때 좋은 질문 리스트를 알려줘. (개인정보 없음)
