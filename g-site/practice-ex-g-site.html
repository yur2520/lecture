구글 사이트로 html 코드 호스팅하기

제미나이가 만든 코드를 구글 사이트 기능을 통해 호스팅할 수 있습니다.

1

구글 사이트 도구에서 [빈 사이트]를 클릭하여 새 프로젝트를 생성합니다.

2

오른쪽 사이드바 패널의 상단에 있는 [페이지] 탭을 선택합니다.

3

하단의 [+] 버튼에 마우스를 올린 후 [전체 페이지 삽입 <>]을 클릭합니다.

4

생성될 페이지의 적절한 이름을 입력하고 [완료]를 누릅니다.

5

생성된 빈 화면 중앙의 [삽입 추가] 버튼을 클릭합니다.

6

팝업창에서 [소스 코드] 탭을 누르고 제미나이의 HTML 코드를 붙여넣습니다.

7

우측 상단의 파란색 [게시] 버튼을 클릭합니다.

8

웹 주소를 설정하고 게시합니다.
